# Instrument-Transposition
IT pipeline within broader ORCA Research

## Repo Layout

- `datasets/inputs/demo`: small demo input dataset used by the notebook harness
- `datasets/inputs/aggregated`: aggregated evaluation inputs, grouped by source
- `datasets/inputs/free_midi_subset`: curated free MIDI subset plus manifest
- `datasets/inputs/free_midi_full`: full free MIDI pack fallback
- `datasets/inputs/uploaded_out.zip`: uploaded MIDI bundle archive
- `datasets/outputs/demo`: generated demo outputs
- `datasets/outputs/aggregated`: generated aggregated outputs and results CSV
